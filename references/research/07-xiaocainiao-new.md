## 新概念/新规则

### 1. 知行交易模块（四层结构总纲）

**来源**: `[202604252223]45加餐我把知行交易模块给你们系统捋一遍` + `[202604160730]450速看交易模块化总复盘` + `[202604160844]451防守是交易的终极哲学`

**完整模块架构**：
- **红色（择时）**: 活跃市值 +4% 多头 / -2.3% 空头 → 所有模块中最重要的第一层
- **蓝色（买入）**: 5 种玩法 — B1、B2、B3、单针下30、砖形图
- **橙色（判断依据）**: 关键K、暴力K、倍量柱、异动、ABC、N型结构、建仓波/拉升波/冲刺波、四块砖、填坑出坑、长阴短柱、筹码理论、量线
- **黄色（应对）**: 没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴 → 破白线 → 破黄线 → 死叉

**核心优先级**：
1. 择时永远第一（天好再干活，天不好就休息）
2. 没涨和盈转亏 > 止损（没涨是受伤前的预防，止损是受伤后的处理）
3. 放飞一定要养成习惯（放飞不是怂，放飞是防守）
4. 盘中只做三件事：盈转亏、关键位置大破位、放飞

### 2. 空头思维四步战法

**来源**: `[202602202212]新春特辑在A股想赚大钱先把自己活成空头波段选股`

核心心法：**把自己定义为空头，用空头思维做多头交易**
- Step 1: 敌不动我不动 — 只看周线B1 + 活跃市值信号，绝不提前入场
- Step 2: 敌进我退 — 拉升就是砸盘信号，谁涨得凶就砸谁（拉20%卖、涨5%再卖20%、涨停砸半仓）
- Step 3: 敌退我追 — 回撤到关键位，拿着钱追着拿带血的筹码
- Step 4: 穷寇莫追 — 连续两次建仓触发止损，直接放弃

### 3. 波段交易三条路

**来源**: `[202602210900]新春特辑在A股我只赚看得懂的钱三种波段七步战法`

| 路径 | 定义 | 难度 | Z哥态度 |
|------|------|------|---------|
| 左侧低估 | 全周期均线空头排列末期，估值跌到极致 | 人性门槛极高 | 老粉可做，新人不碰 |
| 超跌反弹 | 长期下跌后横盘震荡，所有上涨都是反弹 | 最难模式 | **我自己从来不碰** |
| 多头控盘 | 长周期均线多头排列，N型结构已成型 | 确定性最高 | **2026-2027年唯一主战场** |

**多头控盘7步战法**：
1. 只做周线白线之上，N型结构上涨区间
2. 只做已经有大资金布局的标的（底部放量、涨停）
3. 通过周线均线关系判断入场位置（周线B1 或 日线B1）
4. 日线B1信号匹配，周线日线买点共振
5. 周线级别设置均线止损位
6. 日线级别做放飞与卖出（敌进我退，冲高主动砸盘）
7. 趋势没改变，盯住不放，反复操作

### 4. 交易松紧手体系

**来源**: `[202601140059]23交易松紧手`

根据市场状态调整策略松紧度：

**手紧阶段（空头趋势）判断标准**：
- 大盘在黄线之下
- 活跃市值在 -2.3% 波段之内

手紧时的规则：
- 只做最完美形态，"还行"的不要
- 严禁B2、B3，只能做B1
- 仓位 0%-50%
- 原则：市场没证明你是对的就是错的 / 不涨就拍 / 弹起就卖 / 盈转亏必须卖 / 感觉不好也卖

**手松阶段（多头趋势）判断标准**：
- 大盘在白线之上
- 活跃市值 +4% 波段内（单根红柱>4% 或 累计>4%）

手松时的规则：
- 仓位 80%-100%
- B1、B2、B3、暴力K线轮着上
- 核心手法：拿住不动
- 允许SB1加仓

### 5. 五分制持仓评分系统

**来源**: `[202602030001]522赢在起跑线之外一套对抗A股杀猪盘的五维持仓心法`

每天收盘前对持仓打5分：
1. 收盘价是否上升？（今天>昨天 → 1分）
2. 有没有跌破BBI线？（之上 → 1分）
3. 有没有巨量阴线？（缩量/阳量 → 1分）
4. 趋势是否向上？（N型低点抬高 → 1分）
5. 独家指标"勾"是否死叉？（金叉/张口向上 → 1分）

执行：
- 5分：拿住不动
- 4分：继续持有
- 3分：可减半仓，但绝对不能全卖（防止踏空）
- 2分及以下：该止损止损，该离场离场

### 6. 补票战法（深V战法）

**来源**: `[202601060042]12错过起涨点只能踏空聊聊如何在趋势中途安全补票`

核心：在强势股中途洗盘时"补票上车"
- 选股：趋势向上 + 资金背离（红线>80不掉的机构控盘，白线<20的散户恐慌）
- 入场：晚上复盘选出机构高控盘但散户情绪冰点的票，次日早盘出击
- 卖出：一日游心态，不涨即错，止损-2%，脱离成本3%以上减半
- 特点：**胜率不高（~40%）但盈亏比高达1:3**，低胜率高赔率策略

### 7. 交易节奏四维度

**来源**: `[202601190129]334交易节奏的终极心法如何在垃圾时间里守住你的半条命`

四个维度咬合：市场节奏 + 板块节奏 + 个股节奏 + 自己的节奏

资金规模匹配：
- 小资金：学习和练手，快进快出，止损是关键词
- 中等资金：底仓(40%-50%) + 动态仓，死拿会磨死，全动态会累死
- 大资金：跑赢无风险收益率即可

垃圾时间（横盘震荡期）策略：**底仓 + 动态仓**

### 8. 关键K实战心法

**来源**: `[202511100126]关键K实战心法Z哥`

**关键K定义**：在关键位置出现的、能反映主力真实意图、管理后续大部分K线的核心K线

两大核心意义：
1. **趋势反转**（6种）：下跌转上涨 / 下跌转横盘 / 横盘转上涨 / 上涨转下跌 / 上涨转横盘 / 横盘转下跌
2. **走势衰竭**：阶段卖盘枯竭（下跌"加油站"，B1信号）/ 阶段买盘枯竭（上涨"终点站"，S1信号）

核心：位置+量能双重条件，趋势反转是80分的题，走势衰竭是20分的题

### 9. 三最交易心法

**来源**: `[202601210040]353一套穿越牛熊的三最交易心法`

- **只选最美**（买前）：用审视艺术品的眼光去挑剔图形，宁缺毋滥
- **只买最强**（盘中）：用赛马赌徒的直觉去跟随资金，开盘量比爆炸+形态完美就直接干
- **只拿最硬**（买后）：用渣男的心态管理持仓，不涨就换

### 10. 斗牛士战法（牛市持仓心法）

**来源**: `[202602271900]72驯服狂暴如何在牛市中做一名顶级的斗牛士`

- 勇气+决心+技巧：牛市最大敌人不是下跌，是"诱惑"
- B1=挖掘牛（埋伏）/ B2=确认牛（加速）/ B3=不得不发牛（箭在弦上）
- 核心战法：通过"放飞"换取"永久居留权"（爆发减仓，浮盈变负成本后才能真正拿住）
- 建立"ESG 50"或"曼城阵容"（10个左右各行业顶尖公司，反复轮动）
- "一手"练心法：拿100元买一手，练习能不能一直拿到牛市见顶

### 11. 十张死亡K线图（保命底线）

**来源**: `[202603252057]312十张死亡K线图我在股市活了15年的保命底线`

四类死亡图形：
1. 周线空头排列的下跌趋势票 → 绝对不碰
2. 高位黄线跌破必须走 → 最多等一根收回
3. 加速上涨后的大发散（半年涨3倍以上）→ 不接最后一棒
4. 永远不要在下跌趋势里抄底 → 等趋势扭转再进

### 12. MACD一票否决权（升级用法）

**来源**: `[202604230737]470速看从一票否决权到实战博弈全解析` + `[202604231829]471MACD被散户用废的指标之王` + `[202604240812]472从顶底背离到黄线战法`

三大核心用法（外部教材没有的）：
1. **零轴多空判断**：白线(DIFF)上穿零轴=多头 / 下穿=空头 → 做零轴之上的波段
2. **顶背离与底背离**：背离的本质是建仓完成后筹码锁定，不需要再花那么多钱
3. **金叉空与死叉多**：
   - 金叉空：眼看要金叉，白线突然拐头向下 → 下跌趋势延续
   - 死叉多：眼看要死叉，直接跳空高开大涨 → 空中加油

**关键细节**：
- 白线位置不可靠（赌二次金叉）
- **黄线位置交易价值极高**（多头区间打到黄线位置是性价比最高的买点）
- 滞后性是灵魂：正是变化慢，才具备更强的趋势预判性
- 默认参数(12,26,9)，不要改参数

### 13. 量比开盘判断

**来源**: `[202601212203]354B1如何用量比在开盘一分钟看透资金意图`

- 量比 > 20 + 股价红盘向上 → 直接干（爆发模式）
- 量比 10-20 → 观察承接力度（次级选择）
- 量比 < 10 → 放弃第一时间突击（震荡模式）
- 选股占20%，盘中纪律和动态识别占50%

### 14. "扭一扭"本质

**来源**: `[202603250842]311扭一扭里的中国股市生存密码`

- 低位多头排列才有价值，高位多头排列可能是套
- 周线北伐战法：突破长期压制后，要么生拉硬拽（带量上拉），要么时间换空间（横盘磨均线）
- 均线是市场成本线，只有确认突破/跌破后才有意义
- S1信号：宁可信其有不可信其无

### 15. 主题交易三层防火墙

**来源**: `[202604170730]452主题交易的三层防火墙`

在择时正确前提下，五种买入战法 + 应对模块（没涨/盈转亏/止损/放飞/S1/滴滴/破白线/破黄线/死叉）
- **没涨是最高优先级**：市场没证明你是对的，你就错了
- 砖形图第四块红砖后是铁律放飞点
- 黄线之上大哥在，黄线之下大哥不在

### 16. 筹码战争

**来源**: `[202603260753]392筹码战争`

- 底部不需要有量（筹码锁定信号）
- 调仓正确姿势：贾玲换杨幂，不是臭的换更臭的（弱票换成强壮宝宝的低位筹码）
- 2026年已不是2024年闭眼拿住不动的时代
- 事件驱动的大盘股两个涨停 = 小票涨50%-80%，波动率到顶

### 1. B3定义的明确化

**过去**: 仅提及，没有系统讲解

### 2. 止损优先级的重新排序

**过去**: 止损是第一要务
**现在** (`[202604252223]`): **没涨和盈转亏 > 止损**。止损已经是第二步了，真正最重要的是没涨（受伤前的预防）和盈转亏（机会没延续）。做好这两个，一般都不会走到大止损。

### 3. 对MACD滞后性的重新定义

**过去**: MACD有滞后性，是缺点
**现在** (`[202604230737]`): **"新手看滞后，老手看预判。正是因为变化慢，才具备更强的趋势预判性。"** 滞后性只有最后一次，不能因为最后一次的滞后而之前每次都跑。

### 4. 从"拿住不动"到"底仓+动态仓"的进化

**过去** (2024年): 让大家拿住不动
**现在** (`[202603260753]`): "2026年已经到右侧了，估值都高了，不要拿住不动，你拿不住。" 改为底仓(40-50%) + 动态仓策略。

### 5. S1信号的扩展

**过去**: S1=放量阴线
**现在** (`[202603250842]`): "高位放巨量的那根K线，不管是阳线还是阴线，只要在相对高位放了不该放的量，就是预警信号。" 不局限于阴线。

### 6. 选股逻辑：从板块到趋势

**过去**: 可能先看板块/行业
**现在** (`[202602212135]`): "周线选股，不要看板块和概念，没有任何意义。" 趋势已成立的龙头会自己带着板块走，本末倒置挑到的都是跟风杂毛。

### 7. 砖形图的定位进化

**过去**: 可能作为独立战法
**现在** (`[202604252223]`): 砖形图是"再确认"工具，可嫁接在B1/B2/B3/单针下30上。第四块红砖是铁律放飞点，第五/六块进去就是接盘。

## 交易心理

### 1. 知行合一的困境

> "知行合一为什么难？因为你回头看K线，一扒拉，十秒钟就过去了；但你真拿着它的时候，一根K线要熬四个小时，盘后还要被各种消息、自媒体、弹幕折磨。"

### 2. 持仓的道德绑架

> "你买了一只票，套住了，明明知道该止损了，就是舍不得卖，觉得'我都亏了这么多了，卖了不就浪费了吗'——就像硬吃烧麦长胖，为了不浪费沉没成本，最后付出了更大的代价。"

### 3. 前女友心态（踏空痛苦）

> "没买的票涨停了，你无所谓（路人/迪丽热巴）。但你买了被洗下车，眼睁睁看它翻倍，那种难受——就像尾随前女友到万豪酒店门口。这种痛苦会毁了后面所有操作。"

### 4. 恐惧的根源

> "怕，你就会输一辈子。牛市最大敌人不是下跌，是诱惑。一只票涨80%，中间可能8天缩量调整，每天100个涨停诱惑，8天800个。你的心态会瞬间崩塌。"

### 5. 交易的三层境界

1. 看山是山：追涨杀跌，情绪主导
2. 看山不是山：空头思维，掌握主动权
3. 看山还是山：对大周期的信仰，跟国运站在一起

### 6. 精神力管理

> "不要每天在5400只股票里大海捞针，那会耗尽你的精神力。" 建立固定的"曼城阵容"或"ESG 50"，缩小范围。

### 7. 完美主义的陷阱

> "接受不完美。交易永远是一个不完美的过程。你不可能买在最低点，卖在最高点。"

> "你不是在做美巡赛、欧巡赛，你先把练习场的球打直。"

### 8. 小资金与心态匹配

> "小资金体量：如果这笔钱是你的半条命，就别玩了。唯一的出路是学习和练手，或者追求爆发力。"

> "如果你发现自己心理素质不行，只买一手。十万的账户，买15块钱的一手（1500元）。任务不是赚钱，是练习能不能一直拿到牛市见顶。"
